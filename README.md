# PrimeFinder

**PrimeFinder** is an upcoming C++ CLI application that intends to either:
- Find out if a number is prime or not (primality check);
- or find prime numbers up to a certain range (e.g.: a reasonably-sized mersenne prime) using the Sieve of Erastothenes

These operations are expected to be computationally expensive to run as a single-threaded application. To optimise this, implementation of concurrency and multi-threading is expected to be implemented soon.

# Getting Started

This application is intended to be worked on and run on a Windows Environment using **Visual Studio 2022** (at least) and with the **Desktop development with C++** workload. Refer to [Install C and C++ support in Visual Studio](https://learn.microsoft.com/en-us/cpp/build/vscpp-step-0-installation?view=msvc-170) for the full instructions on how to set this up.

As of writing, the application's entrypoint is through Visual Studio 2022 with the **Start Debugging** (Local Windows Debugger) functionality when the solution file is opened. Depending on the requirements defined on this project/assignment, this may open up to an actual binary and potentially have *nix support.
